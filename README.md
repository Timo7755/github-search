A simple web app to search for GitHub users and view their profile summary. Built with React, tanstack query and the GitHub API.

## What it does

Search – Type a username to find GitHub users (with live suggestions as you type).
User card - See avatar, name, bio, and a link to their GitHub profile.
Recent searches – Your last few searches are saved and clickable (stored in the browser).
Dark mode – Toggle between light and dark theme; preference is remembered.

## Skills & tech used

- React
- TypeScript
- Vite
- TanStack Query 
- Tailwind CSS 
- GitHub API
- Debouncing
- Local storage
